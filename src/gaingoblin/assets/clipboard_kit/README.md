# Gain Goblin Layer Kit

Source canvas: 1448 x 1086px

This kit was generated from a single flattened concept image. That means these are practical UI art slices, not true original Photoshop-style layers. The `00_clean_base_estimated.png` file uses inpainting to remove the drawn UI regions and should be treated as an approximate base.

## Folders

- `cropped_components/` — tight rectangular PNG crops for each UI piece.
- `full_canvas_layers/` — each layer on a transparent 1448x1086 canvas so it can align exactly over the source.
- `masked_components/` — rounded-alpha variants for fields, panels, tabs, and buttons.
- `layer_manifest.json` — coordinates and sizes for all slices.

## Suggested app use

Use `00_clean_base_estimated.png` as the themed background/reference.
Use component crops for:
- header clip
- sidebar tabs
- center panel
- summary panel
- metric strip
- table panel
- bottom buttons

Replace these with true transparent art later when final UI art is approved.
